photo-importer
==============

A perl script to handle the import of non-duplicated photos to a given storage directory.  Requires a moderatley recent version of perl and the following:

   - File::Find
   - File::Basename
   - File::Copy
   - File::Path

   - DBI
   - DBD::SQLite
   - Digest::SHA
   - Getopt::Long

   - Image::ExifTool
